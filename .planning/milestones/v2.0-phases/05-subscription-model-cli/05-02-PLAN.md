  <action>
Create `org.saltations.subscription.AddSubscriptionCommand` as `@Singleton`, `Callable<Integer>`, `@Command(name = "add", description = "Add a subscription linking a catalog path into a project workspace path.", mixinStandardHelpOptions = true)`.

Picocli options (exact names):
- `@Option(names = {"--project"}, required = true) String project;`
- `@Option(names = {"--publisher"}, required = true) String publisher;`
- `@Option(names = {"--source"}, required = true) String source;` — relative to publisher root
- `@Option(names = {"-w", "--workspace"}, required = true) String workspace;` — relative to project workspace
- `@Option(names = {"--direction"}, required = true) String directionRaw;` — pass to `SubscriptionSyncDirection.fromCliKebab(directionRaw.trim())` inside `call()`; on `IllegalArgumentException` print to stderr and return 1

Inject `ConfigService` via constructor `@Inject`.

Logic in `call()`:
1. `configService.load()` — catch `ConfigNotInitializedException` → stderr message, return 1 (same pattern as AddMappingCommand).
2. Find project by `project` name; if missing → stderr `Project '<name>' not found.` return 1.
3. Find publisher in `config.getCatalogs()` by name; if missing → stderr `Publisher '<name>' not found.` return 1.
4. **Path guard:** After `trim()`, reject if `source` or `workspace` path contains `".."` as a path segment OR starts with `/` or Windows drive pattern — use a small private method `boolean isUnsafeRelative(String p)` that splits on `/` and `\` and rejects any segment `..` or empty problematic cases; print `Error: invalid path: ...` to stderr, return 1.
5. **Global id uniqueness:** Before add, iterate all `config.getProjects()` and all `proj.getSubscriptions()` — if any `subscription.getId().equals(newId)` regenerate UUID (loop until unique — practically once).
6. **Duplicate triple:** On the **target project only**, if any existing subscription has same `catalogName`, `sourcePath`, `workspacePath` (after trim, exact string match) → stderr `Subscription already exists for this project with the same publisher, source, and workspace path.` return 1.
7. **Publisher path warn:** Resolve publisher `CatalogEntry.getPath()` — if `!Files.exists(Path.of(publisherPath))` → `spec.commandLine().getErr().println("Warning: publisher path does not exist on disk: " + publisherPath);` (do not return).
8. Create `SubscriptionEntry` with `UUID.randomUUID().toString()` for id, trimmed source and workspace path strings, direction from `fromCliKebab`.
9. `proj.getSubscriptions().add(entry);` then `configService.save(config);`
10. Success stdout: `Subscription added (id=<id>).` 
11. Return 0.

Use `@Spec CommandSpec spec` for stdout/stderr like AddMappingCommand.
