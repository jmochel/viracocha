  <action>
Complete one-way sync in **`DefaultSyncService`**:

**Walk:** Collect **regular file** relative paths (normalized `/` strings) under each subtree, skipping hidden per **`PatternPathUtils`**. Sort **lexicographically**.

**`PUBLISH_TO_WORKSPACE`:** For each relative path existing on publisher side: if workspace side missing → **copy** (`Files.copy` with `StandardCopyOption.CREATE`, create parents). If workspace exists: if both regular files and **`Files.mismatch(pub, ws) == -1`** → **skip**; else → **conflict** `CONTENT_MISMATCH` and **do not copy** that path. If publisher has file and workspace has **directory** (or vice versa) → **`TYPE_MISMATCH`**.

**`WORKSPACE_TO_PUBLISH`:** Same logic with source/dest roles swapped (copy workspace → publisher).

**Counters:** Increment `filesCopied`, `filesSkipped`, `conflicts` on `SyncSubscriptionResult` per D-10.

**Tests** in **`DefaultSyncServiceOneWayTest`**: follow **`GeneratorServiceTest`** style — **`@TempDir`**, anonymous **`XdgPaths`** override pointing `configFile()` under temp dir, **`configService.init()`**, **`new DefaultSyncService(configService)`** in `@BeforeEach` (no `@MicronautTest` required unless you register a test factory).

Minimum **3** `@Test` methods:
1. **SYN-01:** `PUBLISH_TO_WORKSPACE` — file only on publisher → appears in workspace; assert `filesCopied >= 1`.
2. **SYN-02:** `WORKSPACE_TO_PUBLISH` — file only in workspace → copied to publisher tree.
3. **SYN-04:** Both sides have same relative path with **different bytes** → `CONTENT_MISMATCH` in result, destination unchanged after analyze (no overwrite).

4. **SYN-05:** Path under subtree with segment `.hidden` or name starting with `.` — file not propagated (assert skipped or absent on other side).

Use **`PublisherEntry`**, **`ProjectEntry`**, **`SubscriptionEntry`**, **`SubscriptionSyncDirection`** with real on-disk publisher/workspace folders under `@TempDir`.
  </action>
